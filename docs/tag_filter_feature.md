# タグ機能の追加提案

## 実装するもの
- タスクに任意のタグを登録できる仕組みを追加
- タグでタスクを絞り込むフィルタ機能をダッシュボードに実装

## 各種機能
### タグの付与
タスク作成・編集時にタグを入力できるフォームを用意し、複数タグを登録可能にします。
タグは色を指定できるようにし、ダッシュボード上のカードにも表示します。

### タグフィルタ
`AllTodoList`上部にタグ一覧を表示する`TagFilter`コンポーネントを追加します。既存の`NaviHeader`と同じスタイルでボタンを並べ、選択したタグに応じて一覧を更新します。

### デザイン方針
- 既存の青系カラーと丸みを帯びたカードデザインを踏襲
- タグ表示はバッジ形式で、`rounded-full`と淡い背景色を使用
- フィルタボタンは`NaviHeader`と同様に`bg-blue-500 text-white`を採用
